# Aslan

Aslan 是一个使用 Flutter 开发的番剧管理与在线观看应用。当前版本已经移除规则系统，改为通过用户配置的兼容数据源服务器获取番剧目录、搜索、时间表、详情与播放地址。

本项目是 Kazumi 的 fork / 修改版本，但不是 Kazumi 官方版本。必要的上游归属、授权与修改声明请参阅 [NOTICE](NOTICE)。

## 数据源

Aslan 不内置公开数据源服务器地址。首次使用前，请在应用内进入：

```text
设置 -> 服务器地址
```

并填入你自己的兼容 API 地址。

为了避免在公开仓库中暴露个人服务，本仓库不会提供默认 API 地址，也不会在代码中回填维护者自用服务器。

## 支持平台

- Android 10 及以上
- Windows 10 及以上
- macOS 10.15 及以上
- Linux（实验性）
- iOS 13 及以上（需要自签名）

## 构建

本项目编译需要良好的网络环境。除了由 Google 托管的 Flutter 相关依赖外，本项目同样依赖托管在 MavenCentral、GitHub、SourceForge 等平台上的资源。如果你位于中国大陆，可能需要设置恰当的镜像地址。

```bash
flutter pub get
flutter test
flutter build apk
```
